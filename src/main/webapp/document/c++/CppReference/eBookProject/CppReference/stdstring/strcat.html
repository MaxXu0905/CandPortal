strcat
Syntax: 
  #include <string.h>
  char *strcat( char *str1, const char *str2 );

The strcat() function concatenates str2 onto the end of str1, and returns str1. For example:

   printf( "Enter your name: " );
   scanf( "%s", name );
   title = strcat( name, " the Great" );
   printf( "Hello, %s\n", title );
