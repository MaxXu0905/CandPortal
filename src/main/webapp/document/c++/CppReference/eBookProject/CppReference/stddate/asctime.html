asctime
Syntax: 
  #include <time.h>
  char *asctime( const struct tm *ptr );

The function asctime() converts the time in the struct 'ptr' to a character string of the following format:

   day month date hours:minutes:seconds year

An example:

   Mon Jun 26 12:03:53 2000
